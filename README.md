## About these files

These dotfiles are managed using chezmoi for my three computers:

- Primary desktop (hostname: deskytoppy (don't ask))
- Primary laptop (hostname: Patricks-MacBook-Air)
- Linux Mac Mini (hostname: MacMini)

I'll detail dependencies and what-have-you below.

## Dependencies

- [Fastfetch](https://github.com/fastfetch-cli/fastfetch)
- [Kitty](https://github.com/kovidgoyal/kitty)

# Linux specific

- [Swayfx](https://github.com/WillPower3309/swayfx) (just [sway](https://github.com/swaywm/sway) works, but some options will need excluding)
- [Swaylock](https://github.com/swaywm/swaylock)
- [Regreet](https://github.com/rharish101/ReGreet) (and, by extension, [greetd](https://git.sr.ht/~kennylevinsen/greetd))
- [Waybar](https://github.com/Alexays/Waybar)

# MacOs specific

- [Yabai](https://github.com/asmvik/yabai)
- [Skhd](https://github.com/asmvik/skhd)
- [Sketchybar](https://github.com/FelixKratz/SketchyBar)
- [Sketchybar Lua Plugin](https://github.com/FelixKratz/SbarLua)
- [Kitty themes](https://github.com/dexpota/kitty-themes) ([this one](https://github.com/elmerceng/kitty-themes) probably works too)
